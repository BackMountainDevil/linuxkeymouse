How to read this docs elegantly except reading md files directly?

Just Run the bash shell, then look at http://localhost:3000  

- Linux  
`bash docs/run.sh`

- Else  
    - python 3  
        `cd docs && python -m http.server 3000`
    - python 2  
        `cd docs && python -m SimpleHTTPServer 3000`
    - docsify  
        `docsify serve docs`
